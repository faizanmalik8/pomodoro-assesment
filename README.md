# pomodoro-assesment
A dynamic timer with focus and rest durations with a full control in user hands
A single-screen Pomodoro timer built with vanilla HTML, CSS, and JavaScript(used AI because i dont know javascript).

## Features
- Configurable Focus (25m) and Break (5m) periods.
- Smooth CSS variable-based theme transitions.
- Daily local storage history that persists on reload but resets on a new calendar day.
- Native browser audio cues.

## How to Run Locally
1. Clone this repository to your local machine.
2. Open the folder.
3. Double-click the `index.html` file to open it directly in any modern web browser.
   *(No `npm install` or build steps required).*
